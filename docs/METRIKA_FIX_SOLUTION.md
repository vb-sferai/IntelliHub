## Почему в панели Метрики показывает нули?

### Причина 1: Фильтрация localhost трафика
Яндекс.Метрика может фильтровать визиты с localhost.

**Решение:**
1. Откройте настройки счетчика: https://metrika.yandex.ru/settings?id=105383668
2. Перейдите в раздел "Фильтры"
3. Отключите фильтр "Не учитывать мои визиты"
4. Добавьте localhost в список разрешенных доменов

### Причина 2: Счетчик не активирован
Новые счетчики требуют активации после создания.

**Решение:**
1. Проверьте статус счетчика в настройках
2. Если есть предупреждение об активации - следуйте инструкциям

### Причина 3: Неправильный домен
Счетчик может быть настроен только для ru.sfer.ai

**Решение:**
1. В настройках счетчика добавьте localhost:* в список доменов
2. Или протестируйте на реальном домене

## Временное решение для локального тестирования

Измените hosts файл для имитации реального домена:

### Mac/Linux:
```bash
sudo nano /etc/hosts
# Добавьте строку:
127.0.0.1 local.sfer.ai
```

### Windows:
```
C:\Windows\System32\drivers\etc\hosts
# Добавьте строку:
127.0.0.1 local.sfer.ai
```

Затем откройте: http://local.sfer.ai:5176/baza
